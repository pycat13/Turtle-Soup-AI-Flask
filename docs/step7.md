# Step 7：聊天布局居中化（参考示例）

## 主要工作
- 让对话区域聚焦在页面中间，两侧留白，不再占满全屏。
- 输入栏、状态栏、操作按钮均居中对齐，聊天气泡依然左右区分。

## 前端改动
- `frontend/play.html`
  - 引入 `.topbar-inner` 包裹标题/返回，保持顶部内容居中。
  - 新增 `.chat-shell` 包裹状态条、情境卡片、聊天区，使主体区域限定宽度。
- `frontend/assets/css/play.css`
  - 新增 `.chat-shell` 和 `.topbar-inner`，最大宽度 960px，左右留白。
  - 聊天区改为居中布局，缩短气泡间距，保持左右分色。
  - `chat-input`、`finish-bar` 定位改为居中（max-width 960px，居中固定），避免填满全屏。
  - 调整状态条、情境卡的内边距与间距，整体视觉更像示例。

## 视觉效果
- 聊天主体、输入栏、按钮均在页面中央，左右有留白。
- 气泡仍按“系统/AI 左侧、用户右侧”显示；顶部状态条与情境卡片宽度随容器居中。***
